# An-lisis-antena-simple---mimo-beamforming-5g
Análisis comparativo del desempeño de Antenas Convencionales y Massive MIMO con Beamforming para redes 5G.  Este proyecto implementa simulaciones en MATLAB para evaluar cobertura, SNR, BER y capacidad de Shannon en sistemas de antena simple y Massive MIMO (8, 16 y 32 antenas), aplicando técnicas de beamforming para redes 5G mmWave.
